## 데이터

모든 데이터는 **브라우저 localStorage**에만 저장됩니다. 페이지는 공개돼 있지만 수집한 레퍼런스·기획 메모는 외부에 노출되지 않습니다.

다른 기기로 옮기거나 백업이 필요하면 각 lab의 **↓ JSON 백업** 버튼을 사용하세요.
